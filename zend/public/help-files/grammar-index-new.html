Add/Import Grammar

The Add/Import Grammar page allows users to either create their own grammar from scratch or import an export grammar file. The page is divided into two sections, and the user fills in the side associated with the action he wants to take.

To create a brand-new grammar:

1. On the left side of the page, enter the name, description, and select an iconset for the grammar you want to create. If you are unsure which iconset you wish to use, select the name of iconset and click on the Preview button to see images from that set.

2. Click on the Create New Grammar button. This will save your new grammar and load the Manage Grammar page so you can continue to refine your new grammar.

To import a grammar, you will need an export file from another grammar. With this file in hand (or on disk):

1. On the right side of the page, click on the Browse button to select the export file from your disk.

2. Click on the Import Grammar button. This will lead to the Import Grammar page, where you will be asked to confirm some of the settings.
