Simulate Design

GenoCAD uses the COPASI simulation engine to allow users to view simulations of designs associated with attribute grammars.

To run a simulation:

1. Enter the length of time (in seconds) for the full simulation in the Duration text box.

2. Enter the interval time (in seconds) between measurements in the Interval Time text box.

3. At this point, the # of Intervals will be calculated automatically, and the Run Simulation button will be activated. Due to time constraints, GenoCAD does not currently support simulations of more than 1000 Intervals.

4. Click on the Run Simulation button.

This will load a graph charting the simulation of your design. Some additional features to be aware of:

  • > By default, all of the data elements are charted on the graph. If you want to remove data elements from the graph, you can click on the Show Data Series dropdown and uncheck data elements you do not want to see on the graph (or, conversely, check data elements you want to add back to the graph.)
  • > If you want to download the output of the simulation (the numbers used generate the graph) to load into another application, you can click on the Download Results button.
  • > If you want to zoom into a particular area of the graph, use the mouse to "draw" a box around the area you want to zoom into (in much the same way you select a box from an image using a Paint program.) When you are ready to zoom back out to the full graph again, just double-click on the graph.
  • > If you want to view the exact coordinates of a particular spot on the graph, hover the mouse arrow over the area you are interested in, and the coordinates will be displayed.
