Copy Grammar

The Copy Grammar page allows the user to make a copy of a selected grammar. The Name, Description, Starting Category, and Icon Set are all required fields, and, with the exception of Name, default to the corresponding value in the original grammar (Name defaults to "Copy of" and the name of the Grammar).

By default, any library associated with the base grammar that the user has access to -- ie, public libraries or libraries defined by the logged-in user -- are also copied with the grammar, including associated parts and designs. However, the user may uncheck libraries that he/she would prefer not to copy.

After the user clicks on the Create New Grammar button, they will be routed to the Grammar Editor with the new grammar loaded by default.
